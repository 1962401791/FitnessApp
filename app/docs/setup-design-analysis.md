## 二、优化方案

1. **4-A 文案**：`heroText1`、`heroPara1` 设为 `width: "fill_container"`，避免溢出
2. **4-A 布局**：内容区加 `justifyContent: "space_between"` 或 spacer，使 Next 贴底
3. **4.5 目标**：为 "Gain Weight" 增加选中态（fill + stroke accent）
4. **4.6 活动**：增大选项高度或增加说明，减少留白
5. **4.7 头像**：确保 `alignSelf: "center"`，相机图标使用相对定位
6. **进度条**：可选，改为按步骤显示进度宽度
